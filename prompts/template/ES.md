# Preparar tu Plantilla/Proyecto

> **Fase 1 de la guia [HIRED! How to Get a Job in 2026](../../guides/GET_A_JOB_EN.pdf):** Construye tu arsenal ANTES de necesitarlo.

```
Quiero preparar una plantilla/proyecto Open Source que pueda reutilizar para pruebas
tecnicas de entrevistas de trabajo. Necesito que me ayudes a crear o personalizar un
template existente como [este](https://github.com/proyecto26/projectx) que demuestre decisiones de arquitectura, no solo codigo.

El template debe incluir:
- Configuracion de testing (unitarios, integracion, e2e)
- Pipeline de CI/CD
- Documentacion profesional (README, ADRs, CONTRIBUTING)
- Buenas practicas del stack que uso: <MY_TECH_STACK>
- Estructura de proyecto escalable y mantenible

Este es un template base que quiero usar/personalizar: <TEMPLATE_URL>

Quiero que el template demuestre juicio profesional y pensamiento de produccion, porque
la IA puede implementar rapido pero YO demuestro las decisiones. Ayudame a identificar
que decisiones de arquitectura y configuraciones debo incluir para destacar frente a
candidatos que solo implementan sin estructura.
```
